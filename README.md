# Awesome Custom Radio Buttons using only HTML & CSS | Option Buttons in CSS

Um projeto simples de **radio buttons personalizados (option buttons)** usando apenas HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar **option buttons modernos**, substituindo o estilo padrão dos radio buttons por um design mais atrativo.

## 🛠️ Tecnologias

- HTML
- CSS

## 🎯 Funcionalidades

- Radio buttons customizados
- Design moderno e interativo
- Sem uso de JavaScript

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
